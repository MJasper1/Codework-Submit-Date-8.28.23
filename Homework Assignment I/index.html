CSS Specificity

Specificity determines which CSS styles should be appled to an HTML element when conficting styles are defined.


What factors influence specificity?

Location (from most to least Specific)

  • Inline (in the HTML element)
  • Internal (in the HTML)
  • External (styelsheet)

Selector (from most to least specific)

  • ID
  • Class
  • Tag name

Order

Styles defined lower in the document override style defined above.
